# BrainRogue v0.1.0-local

First local friends build of BrainRogue.

## Playable Creature Lines

- Tungling -> Sahur Drummer -> Tung Tung Tung Sahur
- Patapim -> Patapim Turbo -> Brr Brr Patapim
- Croclet -> Bombardiro -> Bombardiro Crocodilo
- Trala Kid -> Tralalero -> Tralalero Tralala

## Included Flow

- Main menu with Continue, New Game, Load Game placeholder, Run History, and Settings.
- Starter select with four Stage 1 starters.
- Battle loop with Fight, Ball placeholder, Creature placeholder, and Run confirmation.
- Reward selection after winning a battle.
- Wave progression with boss waves every 10 waves.
- LocalStorage save/continue support.

## Known Issues

- Ball/catching is a placeholder.
- Creature switching is a placeholder.
- Settings shows current volume values and controls info, but does not have sliders yet.
- Audio hooks are implemented, but no final SFX files are included.
- Several release-line creatures still use fallback sprites until final PNGs are added.
- Online accounts, cloud saves, and leaderboards are not included.

## Planned Next Features

- Final sprite pass for all release-line creatures.
- Basic catch flow for Ball.
- Creature switching and small party management.
- Settings controls for volume.
- Additional balancing after local playtest feedback.
